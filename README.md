enable anyone to easily deploy their own blockchain with minimal overhead.</div><a class="button button-simple" href="/#roadmap">See the roadmap</a><a class="button button-white" href="/#join-the-community">Join the community</a></div></section><div class="container"><h2 class="with-decor">Advantages</h2><section class="advantages-section"><div class="row"><div class="col col-12 col-lg-6 pe-md-5"><div class="advantage mb-5 pb-lg-5 pt-lg-4"><div class="row"><div class="col col-12 col-sm-4 pe-sm-3"><div></div></div><div class="col col-12 col-sm-8 ps-sm-3"><div class="title">Deploy a blockchain as easily as a smart contract</div><div class="text">Celestia provides consensus and security on-demand, enabling anyone to deploy a blockchain without the overhead of bootstrapping a new consensus network.</div></div></div></div></div><div class="col col-12 col-lg-6 pe-md-5"><div class="advantage mb-5 pb-lg-5 pt-lg-4"><div class="row"><div class="col col-12 col-sm-4 pe-sm-3"><div></div></div><div class="col col-12 col-sm-8 ps-sm-3"><div class="title">Scaling made simple with data availability sampling</div><div class="text">Because Celestia does not validate transactions, its throughput is not bottlenecked by state execution like traditional blockchains. Thanks to a property of data availability sampling, Celestia’s throughput scales with the number of users.</div></div></div></div></div><div class="col col-12 col-lg-6 pe-md-5"><div class="advantage mb-5 pb-lg-5 pt-lg-4"><div class="row"><div class="col col-12 col-sm-4 pe-sm-3"><div></div></div><div class="col col-12 col-sm-8 ps-sm-3"><div class="title">Secure interoperability for all Celestia chains</div><div class="text">Blockchains built on top of Celestia do not rely on honest majority assumptions for state validity, meaning that they can interoperate with the highest security standards.</div></div></div></div></div><div class="col col-12 col-lg-6 pe-md-5"><div class="advantage mb-5 pb-lg-5 pt-lg-4"><div class="row"><div class="col col-12 col-sm-4 pe-sm-3"><div></div></div><div class="col col-12 col-sm-8 ps-sm-3"><div class="title">Choose your own execution environment</div><div class="text">Since Celestia does not impose any execution logic, developers are free to define their own execution environments.
